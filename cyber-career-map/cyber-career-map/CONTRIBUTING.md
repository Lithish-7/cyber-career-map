# Contributing Guide

Thanks for helping improve the map!

## Add or update a role
- Edit `data/roles.json`
- Keep fields: `title`, `domain`, `summary`, `skills[]`, `certs[]`, `links[{label,url}]`
- Keep skills/certs concise (<= 8 each).

## Quality checks
- Prefer free/official resources.
- Avoid affiliate links.
- Keep tone neutral.

## PR template (copy into your description)
- Why this change is helpful
- Sources for resources you added
- Who might benefit (students, blue team, etc.)

> Tip: If you’re a practitioner, add yourself as a resource and we’ll tag you in the release notes.
